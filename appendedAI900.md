# モデルのトレーニングの流れ


graph TD
    A[データを集める] --> B[データをきれいにする]
    B --> C[勉強前のモデルを選ぶ]
    C --> D[モデルに勉強させる]
    D --> E[賢くなったモデルをテストする]
    E --> F[賢くなったモデルを改善する]

1. **データを集める**
   - 写真や文章を集めるよ

2. **データをきれいにする**
   - 間違ったデータを直したり、抜けているところを埋めるよ

3. **勉強前のモデルを選ぶ**
   - どんな方法で勉強するか決めるよ

4. **モデルに勉強させる**
   - コンピュータにデータを見せて学ばせるよ

5. **賢くなったモデルをテストする**
   - ちゃんと学んだかどうか確認するよ

6. **賢くなったモデルを改善する**
   - もっと良くなるように工夫するよ
